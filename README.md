# Auditor de Caja — Transportes Bello e Hijos Ltda.

Aplicación web para auditar egresos de caja de todas las sucursales.

## Setup

```bash
npm install
npm run dev
```

## Deploy (Vercel)

1. Push a GitHub
2. Importar en Vercel
3. Framework: Vite
4. Build: `npm run build`
5. Output: `dist`

## Uso

1. Exportar desde TMS: módulo Caja → todas las sucursales → rango deseado
2. Subir el .xls directo en la app
3. Revisar alertas por severidad y sucursal
